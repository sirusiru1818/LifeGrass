# 인생 잔디 (LifeGrass)

**"내 인생의 커밋 로그: 한 주를 심고, 내일을 추천받다"**

인생의 유한함을 시각화하고(Memento Mori), 매주 기록을 통해 성장을 지속하는 **라이프 로깅 및 커리어 코칭** 플랫폼.

---

## 서비스 컨셉

- **핵심 가치**: 인생의 유한함을 시각화하고, 매주 기록을 통해 성장을 지속
- **메타포**: 깃허브의 잔디 한 칸 = 내 인생의 일주일

---

## 주요 기능

### 1. 인생 잔디 그리드 (Life Calendar)

- 태어난 날부터 현재까지의 **주차**를 시각화
- 지나간 주는 채워져 있고, 남은 주차는 비어 있는 **잔디밭** 형태
- 각 칸의 색상은 그 주의 **성취도** 또는 **만족도**에 따라 농도가 달라짐

### 2. 위클리 레트로 (Weekly Journaling)

- 한 주간의 주요 활동, 배운 점, 느낀 점을 간단히 기록

### 3. AI 라이프 가이드 (LLM 기반)

- 사용자의 일기를 분석하여 다음 주에 시도해볼 **Personal Challenge 3가지** 추천
- 작성된 데이터가 쌓일수록 사용자의 성향과 커리어 방향에 맞는 정교한 조언 제공

### 4. 마일스톤 설정

- 특정 주차(예: 30세 1주차)에 목표를 설정하고 잔디밭에 **깃발** 표시

---

## 사용자 경험 (User Flow)

1. **입장**: 이번 주 잔디 칸이 깜빡이며 기록을 독촉
2. **기록**: 사용자가 이번 주의 키워드와 짧은 회고 작성
3. **심기**: '심기' 버튼을 누르면 해당 주차의 잔디 색이 변하며 배치
4. **추천**: LLM이 일기를 읽고 맞춤형 챌린지 제시  
   *(예: "이번 주에 운동에 대해 고민하셨네요. 다음 주는 30분 산책 어때요?")*
5. **확장**: 전체 그리드를 축소해서 보며 내 인생이 얼마나 채워졌는지 확인

---

## 기대 효과

- **시각적 자극**: 막연한 시간을 구체적인 칸으로 인식하게 하여 나태함을 방지
- **성장 동력**: AI가 단순 기록에 그치지 않고, 다음 행동(Action Item)을 제안함으로써 실제 변화를 유도

## 프로젝트 설정

1. `.env.example`을 `.env`로 복사한 뒤 필요한 값 입력
2. 의존성 설치 후 프로젝트 실행

```bash
cp .env.example .env
# .env 편집 후
# npm install 또는 pip install -r requirements.txt 등
```
